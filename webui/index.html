📘 Moodle2AffineMD Web UI

Local HTML interface for converting an unpacked Moodle backup into GitHub-ready Markdown and AFFiNE-friendly imports.

Ready.

How to use it

  1. Enter the unpacked Moodle backup folder as the source.
  2. Choose an output folder.
  3. Optionally enable the single-page and ZIP export options.
  4. Click “Start conversion”.

Example: /path/to/unpacked-moodle-backup → /path/to/output-folder
